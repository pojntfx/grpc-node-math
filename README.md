# Felicitas Pojtinger's Math gRPC Node Microservice

A simple math GRPC microservice, for the purpose of learning and evaluating Node and gRPC.

## Features

- Add numbers
- Delete numbers

## Usage

```bash
# Compile the protos
npm run generate
# Run
npm start
```
